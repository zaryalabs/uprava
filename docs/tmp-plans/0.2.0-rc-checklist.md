## Release rule

Never rebuild the same `0.2.0-rc.N` with different source. Every candidate
records its Git SHA, package versions, Core/Web/Node artifact ids and state-slot
procedure. Final `0.2.0` is cut only after the candidate passes the complete
clean-state suite twice: once for the final RC and once for the immutable final
build.
