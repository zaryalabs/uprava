# Русские документы

Эта папка предназначена для русских черновиков, заметок и промежуточных продуктовых/архитектурных документов.

Рабочий процесс:

1. Обсуждаем и оформляем идеи на русском в `docs/ru`, если так быстрее формулировать смысл.
2. Когда документ стабилизировался, переводим его в `docs/en`.
3. После появления английской версии считаем `docs/en` основным источником правды и продолжаем работу преимущественно там.
4. Русская версия может оставаться как черновик, архив размышлений или место для нового русскоязычного обсуждения.

Текущие документы:

- [`cortex-notes.md`](cortex-notes.md) - исходные заметки и идеи.
- [`vision.md`](vision.md) - draft vision продукта.
- [`architecture.md`](architecture.md) - draft архитектуры Core / Node Daemon / Clients.
- [`tech-stack.md`](tech-stack.md) - предварительный технический стек.
- [`product-stages.md`](product-stages.md) - стадии развития продукта.
- [`feature-inventory.md`](feature-inventory.md) - инвентарь придуманных фич и направлений.
- [`design/`](design/) - карта дизайн-направлений и будущие глубокие design docs.
